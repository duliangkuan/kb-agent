📖 使用说明（clone 下来怎么跑）
  1. 配置：复制 .env.example 为 .env，填入硅基流动 key（必填）。TextIn 凭证可选——不填则只支持 txt / md / 直接粘贴文本。
  2. 安装：pip install -r requirements.txt
  3. 启动：uvicorn app.main:app，浏览器打开 http://localhost:8000/ 即本页。
  4. 建库 → 选中 → 上传：左侧新建知识库，点选它，再粘贴文本或上传 txt/md/PDF/Word 入库。
  5. 检索：右侧输入问题，看「命中来源」（含原文片段）+ 逐字流式生成的回答。
  6. 验证语义检索：上传两篇主题不同的文章，搜一个文中没出现的同义词，仍能命中相关文章，即为语义命中而非关键词匹配。
🔌 接入 Agent（把检索能力作为 MCP 工具）
检索能力已封装为 MCP server，可被 Claude Desktop / Cursor / Claude Code 调用，无需“安装软件”，只需登记本服务。

方式一　Claude Desktop / Cursor（路径换成你本机绝对路径）：

{
  "mcpServers": {
    "kb-agent": {
      "command": "python",
      "args": ["D:\\Dev\\kb-agent\\mcp_server.py"]
    }
  }
}

方式二　Claude Code（一行命令）：

claude mcp add kb-agent -- python D:\Dev\kb-agent\mcp_server.py

保存并重启后，Agent 即拥有 search_knowledge_base / list_knowledge_bases / add_text_to_kb 三个工具。对它说「帮我查微信小程序 AI 化有哪些不同观点」，它会自动检索本地知识库并作答。
