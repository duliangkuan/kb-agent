# 运行说明

## 〇、环境要求

- Python 3.10+
- 一个**硅基流动**（SiliconFlow）API key（必填，用于向量化 / 精排 / 生成）
- （可选）TextIn 的 `app_id` / `secret_code`：上传 PDF/Word/图片时才需要；只用 txt/文本可不填

---

## 一、配置 `.env`

复制模板并填写：

```bash
cp .env.example .env
```

打开 `.env`，至少填入：

```ini
SILICONFLOW_API_KEY=sk-你的硅基流动key      # 必填
# 下面两项可选，上传 PDF/Word 才需要
TEXTIN_APP_ID=你的app_id
TEXTIN_SECRET_CODE=你的secret
```

其余参数（分块大小、top_k、相似度阈值等）有合理默认值，一般无需改动。

> 注：`.env` 含真实密钥，已被 `.gitignore` 忽略，**不会进入交付包**；评测者请填入自己的 key。

---

## 二、安装依赖

```bash
pip install -r requirements.txt
```

---

## 三、启动服务

```bash
uvicorn app.main:app
```

看到日志 `DB 初始化完成` 即启动成功。浏览器打开 **http://localhost:8000/**。

---

## 四、使用网页

1. **新建知识库**：左栏输入名称 → 「新建」。
2. **选中知识库**：点一下使其高亮（高亮即“当前库”）。
3. **上传内容**：
   - 文本：填标题 + 粘贴正文 → 「文本入库」
   - 文件：选 txt/md/PDF/Word → 「上传文件入库」（PDF/Word 会自动走 TextIn 解析）
4. **检索**：右侧输入问题 → 回车。
   - 看「命中来源」：每条带标题、相关度分数、**原文片段**
   - 看「回答」：DeepSeek-V3 基于命中内容**逐字流式**生成
   - 勾选「跨全部知识库检索」可跨库；不勾则只搜当前选中的库
5. **管理**：文档/知识库均可随时删除。

---

## 五、接入 Agent（MCP，可选）

**Claude Code**
```bash
claude mcp add kb-agent -- python <本项目绝对路径>/mcp_server.py
```

**Claude Desktop / Cursor**：在其 MCP 配置加入（前端「🔌 接入 Agent」面板有一键复制）：
```json
{ "mcpServers": { "kb-agent": { "command": "python", "args": ["<绝对路径>/mcp_server.py"] } } }
```
保存并重启客户端后，Agent 即可调用 `search_knowledge_base` 等三个工具。

## 常见问题

- **启动报 “未配置 SILICONFLOW_API_KEY”**：`.env` 没填 key，或没在项目根目录启动。
- **上传 PDF 报 TextIn 错误**：未填 TextIn 凭证；可改用 txt/文本入库（核心功能不依赖 TextIn）。
- **检索返回“未找到相关内容”**：知识库为空、或问题确实与库内容无关（系统拒答不编造，属正常）。
